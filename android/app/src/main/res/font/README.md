# Inter Font Files

Place Inter font files (.ttf) in this directory.

Required files:
- inter_thin.ttf
- inter_extralight.ttf
- inter_light.ttf
- inter_regular.ttf
- inter_medium.ttf
- inter_semibold.ttf
- inter_bold.ttf
- inter_extrabold.ttf
- inter_black.ttf

Download from: https://fonts.google.com/specimen/Inter

These files are not committed to Git (.gitignore).
A future Phase A2 task may automate downloading via a Gradle task.
